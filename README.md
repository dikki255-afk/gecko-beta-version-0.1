# gecko-beta-version-0.1
Gecko (Beta) adalah game action-survival top-down seru! Kendalikan monster kadal raksasa mekanis dengan Virtual Joystick di arena grid neon luas. Hindari tembakan peluru kuning dari AI Hunter yang cerdas dan adaptif. Atur jarak pandang sesukamu dengan Kamera Roblox Zoom. Coba versi beta gratis ini sekarang!
